موقع 'سوق الحي' - نسخة HTML/JS بسيطة (نسخة تجريبية)

كيف تستخدم:
- افتح index.html في المتصفح (سيفتح الموقع محليًا).
- الموقع يخزن الإعلانات في localStorage داخل المتصفح.
- يمكنك إضافة إعلان جديد من زر 'أضف إعلان جديد'.

ملاحظات للتطوير:
- لربطه بخادم حقيقي: أنشئ API لحفظ الإعلانات في قاعدة بيانات (مثلاً Firebase أو MongoDB).
- أضف خاصية تسجيل المستخدمين والدردشة الحقيقية.
- ربط مع خرائط Google لعرض إحداثيات البائعين.
